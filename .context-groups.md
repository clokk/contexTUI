# Context Groups

Define file groups for quick context loading. Each `##` heading creates a group.
List files using relative paths from the project root.

**Usage in contexTUI:**
- Groups appear as badges `[group-name]` next to files in the tree
- Press `g` to open the groups panel
- Copy an entire group to clipboard for Claude
